About the Project

 I created this project for my second computer science class: CISC 230: Object Oriented Design. It is a text based, Java game, where the players goal is to find and hug the angry computer science teacher, Dr. Jarvis. This was the final project we did for the class and incorporated many aspects of what we had learned that semester. It combined basic object oriented design concepts such as polymorphism and inheritance with more complicated aspects such as multithreading. Although this project was from one of the earlier computer science course it remains one of my favorites. It taught me a lot about Java and object oriented design in general.
 The game is played entirely in the console. The keys QWEASDZXC are used to control movement. The board is represented by a grid of 10x10 #'s. As the player moves, the #'s are removed from the board. The goal is to find the Angry Jarvis represented by the '?'. The Jarvis moves randomly around the board laying homework traps as he moves. If the player hits a homework trap he cannot move for 5 seconds. Once the player and the Jarvis are in the same space the Jarvis is soother and the game ends.
